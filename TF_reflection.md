Data Flow:

The system starts by creating an Owner. The Owner stores a list of Pets, and each Pet stores its own Tasks. Tasks represent individual pet care activities like feeding or walking. The Scheduler receives the Owner and retrieves all tasks by looping through the owner’s pets. It then sorts the tasks by time and checks for conflicts. Finally, the program prints the organized schedule and any warnings.

<br>

Inspecting 1 Class Method:

I inspected the Scheduler.sort_by_time() method because it plays an important role in organizing the daily schedule. The method first retrieves all tasks from the owner using owner.get_all_tasks(), which collects tasks from each pet. It then sorts the tasks by their time attribute using Python’s sorted() function. This sorted list is returned to main.py, where it is printed as the daily schedule. Inspecting this method helped me understand how the scheduler organizes tasks across multiple pets.
